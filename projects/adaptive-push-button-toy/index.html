Toy Button Accessibility

Overview

This project modifies a children's toy to enhance accessibility for children with limited motor skills, using an external push button for easier interaction.

It enhances the accessibility of a children's toy by modifying its built-in push button into a larger, external push button, allowing children with developmental delays to engage with the toy independently.

Video Demonstration

Design and Description

This modification supports children with Global Developmental Delay and Cerebral Palsy by allowing them to press a large external button to activate toy functions such as lights and sound. It emphasizes safety, durability, and ease of use.

Conclusion

This project proves that simple, thoughtful modifications can make a significant impact in a child's life. By adapting an off-the-shelf toy with a large, external button, it provides inclusive play, improves dexterity, and encourages independent exploration for children with physical and cognitive challenges.
