Bionic Hand using MediaPipe

Overview

This project uses computer vision and Google MediaPipe to detect hand gestures in real-time, translating them into servo motor movements to control a bionic hand. The setup allows intuitive gesture-based control without the need for wearable sensors.

Video Demonstration

Development and Process

The bionic hand consists of servo motors connected to each finger, controlled via an Arduino board. Google MediaPipe is used on a Python-based script to track finger positions. The coordinates are mapped to control signals, enabling fluid finger movement based on real-time camera input.

Initial Prototype — First test with hand tracking and finger actuation.

Components List

  • MediaPipe: For hand gesture recognition
  • Arduino Uno: Microcontroller for servo control
  • Servo Motors (x5): For each finger
  • 3D Printed Hand Frame: Rigid PLA construction
  • Power Supply: 5V battery or USB power
  • Python & OpenCV: Camera capture and data processing

Conclusion

This bionic hand project bridges the gap between human intent and robotic action using vision-based gesture recognition. With further refinement, this system can be extended into prosthetics and remote robotic manipulation, providing intuitive control with minimal hardware.
